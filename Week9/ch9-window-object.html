Ch9: The Window Object

  • Every JavaScript environment has a global object
  • Any variables that are created in the global scope are actually properties of this object
  • and any functions are methods of it
  • In a browser environment the global object is the window object, which represents the browser window that contains a web page.
  • There is no official standard for the BOM, although there are a number of properties and methods that are supported by all the major browsers
  • Every browser window, tab, popup, frame, and iframe has a window object.
  • The window.location property is an object that contains information about the URL of the current page
  • The window.history property can be used to access information about any previously visited pages in the current browser session.
  • Each window object contains a document object. This object has properties and methods that deal with the page that has been loaded into the window
  • Cookies are small files that are saved locally on a user’s computer. They were invented by Netscape as a way of getting round HTTP being a stateless protocol.
  • To create a cookie, you assign it to JavaScript’s 'cookie jar', using the document.cookie property
  • To see the current contents of the cookie jar, simply enter document.cookie
  • To remove a cookie, you need to set it to expire at a time in the past
  • The window.setTimeout() method accepts a callback to a function as its first parameter and a number of milliseconds as its second parameter
